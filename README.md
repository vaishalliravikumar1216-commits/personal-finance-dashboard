\# 💰 Personal Finance Dashboard



An interactive dashboard built with Python and Streamlit to track and 

visualize personal expenses across categories and months.



\## Features

\- Filter by category and month

\- KPI summary cards (total spent, average, highest)

\- Pie chart for spending distribution

\- Monthly trend bar chart

\- Category-wise breakdown

\- Raw data table



\## Tech Stack

Python | Pandas | Plotly | Streamlit



\## How to Run

pip install -r requirements.txt

streamlit run app.py
